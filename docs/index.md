## Overview

This integration connects your Saunier Duval thermostat (controlled via the MiGO app) to Home Assistant using the Netatmo API.

### Requirements

- Home Assistant 2024.1.0 or later
- A Saunier Duval thermostat compatible with the **MiGO** app
- Valid MiGO app credentials (email and password)

### Supported Devices

- Saunier Duval Isotwin Condens 25-A
- Other Saunier Duval thermostats using the MiGO app (not MiGO Link)
